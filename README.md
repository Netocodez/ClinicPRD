# Project Requirements Collection Form

A professional and responsive HTML form for collecting Software Project Requirement Documents (PRDs) from clients.

This form helps software developers, freelancers, and agencies gather detailed project requirements before development begins.

---

## Features

- Professional modern UI
- Responsive Bootstrap 5 design
- Client information collection
- Project overview section
- Features & functionality requirements
- Design preferences collection
- Technical requirements section
- Budget and timeline inputs
- Formspree integration for backend-free submissions
- Easy deployment on GitHub Pages, Netlify, or Vercel

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Formspree

---

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/Netocodez/ClinicPRD.git
```

---

### 2. Open the Project

Open the project folder in your code editor.

---

### 3. Configure Formspree

Create a free account on:

https://formspree.io

Create a new form and copy your endpoint.

Replace:

```html
action="https://formspree.io/f/YOUR_FORM_ID"
```

with:

```html
action="https://formspree.io/f/your-real-form-id"
```

---

### 4. Run the Project

Simply open:

```bash
index.html
```

in your browser.
